Introduction

What is Bitcoin?

Bitcoin can refer to two things, Bitcoin the software and bitcoin the currency. Bitcoin software is an electronic payment system which uses a specialised currency, bitcoin (or BTC), for all transactions. BTC is a purely digital currency, with no physical form. The Bitcoin system is made up of multiple parts, mainly: nodes, the block chain, and transactions between users.(10), (28)

Nodes

The Bitcoin system runs across multiple computers, creating a peer-to-peer network. Each peer in this network is a node in the system. Across the network (i.e. in the nodes), all of the data on any transactions occurring in the system is stored.(9)

The Block Chain

This network hosts all of the data on transactions which occur in the system. All transaction data is stored in a special data structure, the Block Chain. Data on transactions is added to the block chain in large sections (i.e. blocks) sequentially. This means that to modify past transactions, a user must modify all successive blocks too, as each block stores the hash of the preceding block.

When transactions are made in the network, they are broadcast to other nodes. Some of these nodes (called “miners”) verify the transactions, and if they are valid the miners add them to the block chain. The incentive to do this is the transaction fee, an amount of money given to the miner who adds the transaction to the block chain. Adding a transaction to the block chain requires a substantial amount of work from a miner, so users must rely on them to add their transactions to the block chain (as opposed to doing so themselves).(2), (9)

Transactions

BTC are not technically stored in the system, but instead all BTC spent must come from a transaction. This means that a transaction’s inputs can only be other transactions’ outputs. Because of this, transaction data must be stored in the network (the block chain) and must be publicly visible.

This doesn’t mean that the Bitcoin system has no anonymity, as each transaction doesn’t store any information about its creators directly. Instead, each transaction stores a script which will return true only for the correct user (covered in the verification section). Thanks to this, all transactions are anonymous and secure.(1), (2), (8), (9)

Benefits of Using Bitcoin

Bitcoin has numerous benefits, which are the main reasons for using it:(10), (11), (28)

  • Bitcoin is completely anonymous (i.e. transactions don’t store personally identifying information)
  • Transaction fees paid to miners are lower than those paid in most bank transactions
  • It’s easy to use, and many stores/services accept BTC
  • It’s fast (and important payments can be sped up by increasing the transaction fee)
  • It’s not controlled by a central authority, but by the entire network (so there is no dependency on single nodes of the network, and major changes can be rejected by the users)
  • It can be used internationally and isn’t tied to any one country

Problems with Bitcoin

Bitcoin also has a number of issues:(11), (12), (13), (28)

  • BTC is extremely volatile as a currency, so any payments you receive in BTC could be worthless in a year’s time
  • If a user loses their private key, i.e. their password for the Bitcoin system, all of their BTC can be lost or stolen. Transactions are irreversible compounding this problem
  • Many countries have introduced litigation which opposes the use of BTC, so it can’t be used everywhere legally
